# 📐 Denklem - Eğitsel Matematik Aracı

Ortaokul ve LGS öğrencileri için **interaktif denklem çözücü** aracı!

## 🎯 Özellikler

✨ **Sınıf Seviyeleri:** 5-6-7-LGS (otomatik zorluk seviyeleri)  
⚖️ **Terazi Metaforu:** Denklemin her iki tarafını görselleştirme  
🟦 **Karolı Gösterim:** Bloklar ile matematiksel gösterim  
📍 **Sayı Doğrusu:** Çözümün görselleştirilmesi  
📊 **Adım Adım Çözüm:** Her işlem açıklanmış  
🎬 **Otomatik Çözüm:** Hızlı gösterim modu  
📥 **İndir:** Çözümü metin dosyası olarak indir  
🌙 **Açık/Koyu Tema:** Kullanıcı tercihine göre  

## 🚀 Nasıl Kullanılır?

### Çevrimiçi (GitHub Pages)


## 📝 Örnek Denklemler

| Sınıf | Örnek | Çözüm |
|-------|-------|-------|
| 5. Sınıf | `x + 5 = 12` | `x = 7` |
| 6. Sınıf | `2x + 3 = 11` | `x = 4` |
| 7. Sınıf | `2x + 5 = 3x - 2` | `x = 7` |
| LGS | `(x + 3)/2 = 5` | `x = 7` |
